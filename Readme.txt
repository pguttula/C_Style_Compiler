This folder has a compiler implemented in phases:
1) Folder Lex_And_Yacc has implementation of Lex and Yacc files only.
2) Folder Interpretation_of_AST has code to build an abstract syntax tree for all the operations of a compiler and to interpret it.
3) Folder Interpretation_and_Type_Checking has code to implement static checking.
4) Folder Full_C_style_Compiler- Complete implementation of Compiler with type checking , data flow analysis and interpretation.
